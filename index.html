Projects

  • Dota 2 Social App

    Personally built a web app that allows a user to make an account and login. Users are then able to track different characters and items in the game, Dota 2, using the Steam API. They can create custom hero item builds and save those builds. Users can make posts, comments on any post. A user may also add any build that a player has made to a post to showcase or reference. Built in Ruby on Rails and Vue.js.

  • Bootleg Hulu

    A Next.js app connected to the TMDB API and using Tailwind CSS. You can go on the site and sort the top 20 movies by genre. Will be adding more features soon. Had some issues figuring out deployment to Vercel because of eslint and yarn issues, but managed in the end!
